# Инструкция для работы с Markdown

## Выделение текста

Для выделения текста жирным, обрамить текс двумя звездочками (**), пример **введем текст**. Так же можно обрамить текстк двумя нижними подчеркиваниями (__), пример __введем текст__. 

Для выделения текста курсивом, обрамить текст одной звездочкой (*), пример *тоже текст*. Так же можно обрамить текст одним нижним подчеркиванием (_), пример _тоже текст_.

Несколько способов выделения текста полужирным и курсивом необходимо для одновременного написания текста как полужирным, так и курсивом - _введем одновременно текст курсивом и **полужирным**_
